# 前言

Java语言已经发展了10多年，根据市场反馈，各行各业对Java语言的热度仍然不减，排行上仍然稳居第二，市场上对Java语言学习的热潮仍然没有退去，本项目聚集多个Java开发和架构高手，一起研究和总结JVM的核心技术和关键特性，每个知识点都配以案例和代码，并以文章的形式呈现给大家，最后整理成书，帮助更多的学习Java语言的小伙伴快速掌握Java的要领。
